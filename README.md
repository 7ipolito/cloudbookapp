## Installation

Before you start, ensure you have `npm` installed on your machine. 

1. **Clone the repository**:
   
   ```bash
   git clone https://github.com/7ipolito/cloudbookapp.git
   ```
2. **Install the dependencies**:

     ```bash
     npm install
     ```
